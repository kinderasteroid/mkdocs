# CLI Reference

Command-line tools provided by `qwodel`.

## In this section

| Page | Command | Description |
|---|---|---|
| [check](check.md) | `qwodel check` | Verify installed backends and their dependencies |
| [quantize](quantize.md) | `qwodel quantize` | Quantize a model from the command line |
| [list-formats](list-formats.md) | `qwodel list-formats` | List available quantization formats |
| [list-backends](list-backends.md) | `qwodel list-backends` | List registered backends |

## Usage

```bash
qwodel [COMMAND] [OPTIONS]
```
